# DataHub · JSONPlaceholder Explorer

Aplicación web desarrollada con **Vite + Vanilla JS** que consume
datos de la API pública JSONPlaceholder.

## Páginas

| Página | Descripción |
|--------|-------------|
| Home | Bienvenida al proyecto |
| Posts | Tabla con 100 publicaciones |
| Todos | Tabla con 200 tareas y su estado |
| Usuarios | Directorio con 10 usuarios |
| About | Descripción del proyecto |

## Tecnologías

- Vite
- Vanilla JavaScript
- W3.CSS
- JSONPlaceholder API
